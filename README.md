# 🍎 Fruits Match Game
A Candy Crush-inspired game built using Flask, HTML, CSS, and JavaScript.

## Features
- 8×8 game grid
- Fruit images instead of colored blocks
- Match-3 mechanics
- 🎯 Score tracking
- 📈 Five levels
- 🔄 Dynamic fruit refill
- 🏆 Game completion screen

## Installation
```bash
git clone https://github.com/Nishtha-agarwal/Fruits-game.git
cd Fruits-game
pip install -r requirements.txt
python app.py
