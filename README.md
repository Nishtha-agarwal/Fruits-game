# Candy Crush Game

A Candy Crush-style game built with Flask, HTML, CSS, and JavaScript.

## Features
- 8×8 game grid
- Fruit images instead of colored blocks
- Match-3 mechanics
- Score tracking
- 5 levels
- Dynamic fruit refill
- Game completion screen

## Tech Stack
- Flask
- HTML
- CSS
- JavaScript

## Run Locally

```bash
pip install -r requirements.txt
python app.py
